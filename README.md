# Webpack 4 Quickstart For Vue
### Based on the following
* webpack 4
* typescript
* vue
* vuex
* vue-router
* axios
* element-ui
* pug
* scss

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
