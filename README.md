# Webpack 4 Quickstart For Vue

### Based on the following

* webpack 4
* typescript
* vue
* vuex
* vue-router
* axios
* element-ui
* pug
* scss

### Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm start
```

### Compiles and minifies for production

```bash
npm run build
```
